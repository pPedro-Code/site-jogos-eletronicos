Jogos Eletrônicos

Sobre o projeto

Como os jogos eletrônicos chegaram ao seu estado e como ele afeta a sociedade contemporânea?

Este projeto surgiu incialmente como um trabalho escolar, cujo um dos objetivo era entender melhor a tecnologia e seus possíveis problemas, entre eles, se encontrava os jogos eletrônicos, que é desenvolvido nesse artigo. Com tudo, de inicio, foi feito um projeto apenas com o intuito de relacionar os jogos eletrônicos, o consumismo e consequentemente a economia, entretanto, para melhor compreensão desse "mundo" como um todo, foi feito pesquisas aprofundadas, chegando aos tópicos que serão desenvolvidos. Inclusive, caso queira ver os projetos anteriores, role está página até o final.
